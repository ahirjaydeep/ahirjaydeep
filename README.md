## `> whoami`

I'm **Jaydeep Ravibhai Ahir** — who ships production code, not slide decks.

I build **AI-powered mobile and backend systems** software engineer who learns by building scalable, real-world products.

My focus lies at the intersection of AI, mobile development, and modern system architecture. I enjoy understanding systems end-to-end—from designing clean backend architectures and integrating AI agents, to building intuitive UIs and deploying robust platforms.

As a learner, I don't just write code; I focus on the big picture. Whether it's architecting a SaaS platform, prototyping autonomous systems, or leading a project from an idea to a hackathon podium, I am driven by the entire process of researching, building, and polishing. > **A genuine ADAS prototype with custom YOLO11, real-time lane detection, PID steering, and hardware-enforced failsafes — with a published research paper on Zenodo.**

<table>
<tr>
<td width="50%">

**What it does:**
- Custom-trained YOLO11 model detects persons, stop signs, traffic lights, speed zones
- Dual-lane detection pipeline (CLAHE → Adaptive Threshold → PID steering)
- ESP32 spine with ultrasonic radar brake and Wi-Fi watchdog
- Tank-style differential steering via UDP motor commands
- **Published pre-print**: [zenodo.org/records/19689462](https://zenodo.org/records/19689462)

</td>
<td width="50%">

**Engineering highlights:**
- **Brain/Spine separation** — Python AI on Mac M3, C++ real-time control on ESP32
- Hardware failsafes **cannot be overridden by software bugs**
- kornia_rs (Rust) JPEG decoder for 2× faster frame processing
- PID controller with anti-windup clamping (kp=0.20, kd=0.05)
- Custom fine-tuned model: `zenity_master.pt` (6 classes)

</td>
</tr>
